# self-learning-analytics-portfolio

Tools used: Power BI, Tableau, SQL, Excel
Skills practised: Data cleaning, modelling, DAX, calculated fields, dashboard UI, KPIs, trend analysis.

Projects included:

Coffee Shop Sales Dashboard
Sales trends, product popularity, customer behavior. --> https://github.com/vishal1215/Coffee_shop_analysis

HR Analytics Dashboard
Workforce composition, performance and retention insights. --> https://github.com/vishal1215/HR-DASHBOARD

BikeStores SQL + Tableau Report
SQL queries + retail performance visualisation. --> https://github.com/vishal1215/Bikestores_tab_sql

Plant Performance Report
Simple KPI summary dashboard based on operational data.--> https://github.com/vishal1215/Plant_performance_report

Airways Satisfaction Review
Customer satisfaction trends across airline companies.--> https://github.com/vishal1215/Airways_review_Tableau
